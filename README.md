# MiniGo

  ## Description 
  
  A 5x5 Go playing agent implemented using minimax algorithm with alpha-beta pruning

  ## Table of Contents
  * [Usage](#usage)
  * [License](#license)
  
  ## Usage 
  
  1. Modify the build.sh file to change number of games and opponent agent.
  2. Then run the build.sh file
